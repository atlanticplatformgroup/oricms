# Plugin Manifests and Policy

Use this page for the public plugin manifest and policy seam.

## Stable Surface

- project-repository manifests under `plugins/`
- declared capabilities
- declared hooks
- project enablement and policy
- hook endpoint configuration and secrets

Current manifest file locations:

- `plugins/*.yaml`
- `plugins/*.yml`
- `plugins/*/plugin.yaml`
- `plugins/*/plugin.yml`

## Intentionally Limited

- remote installation
- plugin dependency management
- broad package marketplace semantics
