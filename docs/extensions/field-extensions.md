# Field Extensions

Use this page for the public field-extension seam.

## Stable Surface

- field renderer registration by type
- optional presentation mode
- shared capability resolution for browse, sort, search, and readonly behavior

## What To Rely On

- registry-based resolution
- stable field registration ids
- shared field capability contracts

## What Not To Rely On

- arbitrary runtime loading of third-party field bundles
- undocumented internal editor composition details
