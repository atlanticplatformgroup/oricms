# Backend Lifecycle Hooks

Use this page for the public lifecycle hook seam on the backend.

## Stable Surface

- `before*` and `after*` lifecycle families for current domain operations
- blocking through lifecycle hook errors on `before*` events
- project-scoped payloads with typed event names

## Current Event Families

- entries
- collections
- schemas
- content types
- assets
