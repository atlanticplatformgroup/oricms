# Plugin Operations Troubleshooting

Use this page when plugin webhook delivery or runtime policy behavior is not working as expected.

## Common Checks

- confirm the plugin manifest exists and is valid
- confirm the plugin is enabled for the project
- confirm the hook is declared and configured
- confirm secrets and endpoint URLs are current
- review retry and audit behavior
