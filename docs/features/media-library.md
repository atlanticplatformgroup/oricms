# Media Library

The media library is where a project’s uploaded assets live, along with the metadata editors need to use those files well.

It is not just a file bucket. In a healthy project, it becomes the reuse layer for images and other media referenced throughout the content model.

## What The Feature Does

- assets are project-scoped
- asset metadata is editable in the workspace
- media and image fields resolve through the shared field capability layer

## Typical Workflows

- upload files
- edit metadata
- filter and browse assets
- reuse assets in content fields
