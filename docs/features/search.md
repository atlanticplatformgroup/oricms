# Search

OriCMS includes built-in content search and can also feed a dedicated search index.

## What The Feature Does

- collection search works over entry data and shared field capability semantics
- the CMS is not a full production search engine by itself
- serious public site search may still need external indexing

That split is intentional. OriCMS should be good at helping teams find and deliver structured content. It does not need to impersonate a specialized search engine once ranking, typo tolerance, or facets become important.
