# Getting Started

Start here if you are new to OriCMS.

This section is written to get you from zero to a working project without asking you to learn the whole system up front.

Recommended order:

1. [OriCMS Quickstart](./quickstart.md)
2. [Core Concepts](./core-concepts.md)
3. [Create Your First Project](./first-project.md)
4. [First Publish](./first-publish.md)
