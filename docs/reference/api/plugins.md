# Plugins API

Use the plugins API for plugin catalog, configuration, UI policy, secrets, and event/audit views.

## Current Surface

- catalog and per-plugin lookup
- enablement and reconcile flows
- hook endpoint configuration and retry policy
- execution policy and UI policy
- secret rotation and revocation
- event history and runtime views
