# Assets API

Use the assets API for project-scoped asset reads, uploads, metadata updates, and deletion.

## Current Surface

- raw asset binary reads
- single-asset reads
- asset listing and filtering
- asset upload
- metadata update
- deletion

## Notes

- metadata updates are a separate route from upload
- filtering currently includes folder, tag, usage, search, sort, limit, and offset
- raw asset responses are for binary delivery, not metadata inspection
