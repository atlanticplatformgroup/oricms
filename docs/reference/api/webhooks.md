# Webhooks API

Use the webhook routes for provider-originated events that create build records and trigger mapped environment actions.

## Current Surface

- generic inbound project webhook
- provider-specific webhook endpoints
- build queue handoff
- mapped deployment and revalidation triggering
- provider secret or signature validation where configured
