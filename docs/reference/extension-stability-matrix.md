# Extension Stability Matrix

This page summarizes which extension seams are safe to build against today.

## Stable Enough For Repository-Level Extension

- field renderer registration
- field capability resolution
- workspace section and slot registration
- plugin manifest reading
- lifecycle hook dispatch
- plugin webhook dispatch
- Astro adapter package
- Next.js adapter package

## Current But Curated

- curated OpenAPI snapshot
- client SDK docs and surface

## Intentionally Evolving

- generalized third-party plugin discovery
- versioned extension compatibility negotiation
- full marketplace-style extension packaging
- broad public admin-shell plugin API
