# Integration Support Matrix

This page summarizes the support level of current integration paths.

## First-Class

- Astro
- Next.js

Requirements for first-class status:

- maintained adapter package
- current docs page under `docs/integrations`
- package-level tests
- at least one runnable or copyable example for the primary integration path

## Current But Limited

- curated CLI export paths
- compatibility loaders for legacy repo shapes

## Not First-Class

- removed or unmaintained adapters
- hypothetical framework support without package, docs, tests, and an example path
