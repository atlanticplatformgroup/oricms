# Adapters

Framework adapter packages for consuming OriCMS content in frontend runtimes.

## Present in This Checkout

- `astro`
- `nextjs`

## Current Rule

Adapters are framework-facing packages, so each adapter should carry its own README with:

- install instructions
- minimal usage
- the current expected OriCMS repo shape

Use the per-package READMEs and integration docs under `/docs/integrations/` as the primary guidance.
